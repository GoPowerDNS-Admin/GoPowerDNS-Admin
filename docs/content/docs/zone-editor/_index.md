title: Zone Editor
description: "Manage DNS zones and records in the GoPowerDNS-Admin zone editor, with automatic PTR handling for reverse zones."
weight: 2
---

The zone editor is the central part of GoPowerDNS-Admin. It provides a full-featured DNS record management interface powered by Alpine.js — changes are staged client-side and submitted as a batch, with no full page reloads until you save.
